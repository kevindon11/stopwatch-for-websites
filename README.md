# stopwatch-for-websites

Chrome extension that tracks how long you spend on selected sites each day. It
counts only when a tracked tab is active and optionally shows an on-page overlay
timer.

## Usage

1. Open `chrome://extensions` in Chrome.
2. Enable **Developer mode**.
3. Click **Load unpacked** and select this folder.
4. Pin the extension (puzzle icon â†’ pin).
5. Open the extension popup to edit your tracked site list and toggle the
   overlay.

Tracked times reset per local day. Use the **Reset today** button in the popup
to clear today's totals.
