Site Stopwatch Options

Configure tracked sites and set per-site time and tab limits.

Column legend and defaults

  • Site: Domain or domain/path to track.
  • Daily limit (min): Empty = no daily limit. 0 = always blocked.
  • Break after (min): Minutes of use before a break cycle starts. Empty = off.
  • Block for (min): Break duration once the break cycle is triggered. Empty = off.
  • Entry wait (min): Delay after opening a tracked site before it becomes usable. Empty = off.
  • Edit wait (min): Wait before this site row can be edited again after a change. Empty = off.
  • Tabs: Maximum allowed open tabs for that site. Empty = no tab limit.
