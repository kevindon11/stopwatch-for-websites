Site Stopwatch Options

Configure tracked sites and set per-site time and tab limits.

Site Domain or specific path to track.
Tabs Maximum open tabs for this site. Empty = no tab limit.
Daily limit (min) Max minutes per day. Empty = never block. 0 = block instantly.
Break after (min) Minutes of use before a break starts. Empty = disabled.
Block for (min) Length of each forced break. Empty = disabled.
Entry wait (min) Delay after opening before site becomes usable. Empty = no entry delay.
Edit wait (min) Lock time after edits so values cannot be changed. Empty = editable anytime.
